# 🚦 Traffic Congestion Prediction System (ML Project)

A Machine Learning project that predicts traffic congestion levels using real-world traffic data.  
The system uses historical traffic patterns to classify congestion severity and help in smart traffic management.

---

## Project Objective
To build a predictive model that can:
- Analyze traffic data
- Detect congestion patterns
- Predict traffic intensity levels
- Support smart city traffic optimization

---

## 🧠 Machine Learning Approach
- Data preprocessing and cleaning
- Feature selection
- Model training using classification algorithm
- Model evaluation and prediction

Algorithm used:
- Random Forest Classifier
